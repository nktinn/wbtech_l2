Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Output:
[3, 2, 3]

Объяснение:
    В GO слайс передается как копия. "Подкапотная" реализация слайса включает в себя три поля: указатель на массив, длину и емкость.
Таким образом, при передаче слайса в функцию, копируется указатель на массив, тем самым, все изменения в массиве повлияют
и на оригинальный слайс. Однако, если в функции происходит изменение длины и оно превышает емкость, то создается новая
структура слайса, которая включает в себя новую длину, емкость и новый указатель на массив в памяти, в который копируется
все содержимое оригинального массива.
    В данном случае, изменение i[0] = "3" происходит еще в оригинальном массиве, т.к. i не был пересоздан. После этого
вызывается append(i, "4"), что приводит к созданию нового слайса, в который копируется содержимое оригинального слайса
(т.к. без явного указания емкости в слайсе при инициализации, емкость устанавливается равной длине слайса). После этого
последующие изменения происходят уже в новом массиве и в новом слайсе, что не влияет на оригинальный слайс.
```
