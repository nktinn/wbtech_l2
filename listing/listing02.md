Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Output:
2
1

Объяснение:
    Функция test() возвращает значение x, которое увеличивается на 1 в defer'е. 
    Функция anotherTest() возвращает значение x, которое не изменяется в defer'е.
    
    Это происходит из-за того, что переменная x в первом случае явно указана в сигнатуре функции, а во втором случае - нет.
Из-за этого в первом случае возвращается значение переменной x, которое увеличивается на 1 в defer'е, а во втором случае
все изменения, которые произойдут в defer'е, не будут влиять на возращаемое значение функции.
    
    Defer'ы вызываются в порядке обратном порядку их объявления, после выполнения всех операторов в функции, 
перед возвратом значения.

```
